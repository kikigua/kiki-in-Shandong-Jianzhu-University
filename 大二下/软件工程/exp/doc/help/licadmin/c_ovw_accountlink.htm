About AccountLink

Rational AccountLink is a Web-based tool that manages your permanent license keys for products on Windows, Linux, and the UNIX system.

To get permanent license keys, you provide your account number, product, and host information. Based on the information that you provide, Rational creates permanent license keys for the floating license server or node-locked client system. AccountLink will then send the permanent license file to the contact e-mail address specified in the transaction.

You can specify only one host in each transaction. This means that you can register multiple products to one host or system in a single transaction, but you cannot register multiple products to multiple hosts in a single transaction.

The AccountLink Web site does not support temporary license key transactions.

Note: At the time of publication, AccountLink is the online license fulfillment tool for IBM Rational product customers. In the future, AccountLink will be replaced by the Rational License Key Center, an online system that offers additional features and an easier license key fulfillment experience. You will access the new system, when it is available, using the same IBM site on the Web that is currently used for AccountLink: http://www.ibm.com/software/rational/support/licensing/. This page will provide all of the information and instructions that you need to manage license keys using the Rational License Key Center.

The AccountLink Web site offers three transactions:

Transaction
Explanation or Task
Get license key(s)
Request your permanent license key by registering your IBM Rational product to a specific computer. To order a key, you need the License Key Certificate and the hostname and host ID of the license server or desktop computer.

This transaction will result in a license file being sent to the e-mail address that you specify during the transaction. That license file will contain all of the appropriate license keys for the host you specify during the transaction and any other license keys that have been previously registered to that machine.

  • To find the hostname and host ID, use the tool either in the License Key Administrator or in AccountLink. See Finding the hostname and host ID of a computer.
  • To order license keys for a redundant server environment, enter the redundant server order.
Return license key(s)
Return the license key for the old computer. This transaction updates the IBM Rational records to indicate that you are no longer using the software on the old system.
Note: To move a license key from one machine to another, the license keys must first be returned, then the "Get license key(s)" option can be used to register that license key or keys to the new machine.
For more information about returning or moving license keys, see Moving a license key.
Request a copy of latest license file for a host
Get a copy of a license key file. You may need this transaction when you reformat your hard disk, restore the software on your computer, or your license keys become corrupt.

If you have not yet registered any licenses to a specific host, you must first use the "Get license key(s)" option.
