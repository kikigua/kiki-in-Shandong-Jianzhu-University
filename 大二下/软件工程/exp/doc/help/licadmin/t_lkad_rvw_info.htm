Reviewing license information

You can review information on node-locked, floating, and served licenses on the system that you are using.

By default, you should see complete information about your license keys in the main window of the License Key Administrator. If you do not see this information in the window, use the Show menu item to display information about your licenses.

To display information about your licenses:

  1. Click View > Show.
  2. Select one or all of the following:
    Option Description
    Invalid and Expired Licenses Shows all expired licenses on the system that you are using.
    Floating Licenses Shows all floating licenses on the system that you are using.
    Node Locked Licenses Shows all node-locked licenses on the desktop that you are using.
    Served Licenses Shows all floating licenses that IBM software on your system could request from the license server.
    A check next to the menu item indicates that the option is selected.
