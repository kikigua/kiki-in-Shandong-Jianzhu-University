Changing license usage order

Use the Rational License Key Administrator on your client computer to change the order in which your products use license keys.

To change the license usage order:

  1. Select Settings > License Usage Mapper in the main menu bar.
  2. In the License Usage Mapping window, highlight the product in the Installed Products list.
  3. Move the licenses to your preferred order in License Usage Order.
  4. Click OK.
