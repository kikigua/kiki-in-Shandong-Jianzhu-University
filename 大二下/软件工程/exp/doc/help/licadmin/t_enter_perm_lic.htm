Entering permanent license information

You can configure a permanent license key for your IBM Rational product without a Rational license server.

To operate IBM Rational software purchased by your company:

  1. Go to AccountLink to request a permanent or TLA license key.
  2. Depending on the type of license:
    • Import the node-locked license key file on your desktop.
    • Import the floating (named-user floating) license key file on the Rational license server.
