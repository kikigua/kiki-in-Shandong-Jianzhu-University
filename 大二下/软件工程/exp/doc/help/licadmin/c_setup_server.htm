Configuring license servers

If you are using floating licenses, you have the choice of configuring a single license server, multiple license servers, or a redundant server environment.

  • Using floating and named-user floating licenses
    You can configure the license server to use floating or named-user floating licenses.
  • Setting up the Rational license server
    With floating and named-user licenses you can manage and maintain licenses on single, multiple, or redundant license servers. Use the information in the following topics to help you set up IBM Rational license servers.
  • Configuring named-user floating licenses
    For certain IBM Rational products, you may want to configure named-user floating licenses.
  • Starting the license server
    After you enter the temporary license information or import the floating license file, start the license server.
  • Entering port values on license servers
    In some cases, you may need to change the port values on your single, multiple, or redundant license servers.
