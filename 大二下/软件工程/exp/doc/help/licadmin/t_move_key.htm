Moving a license key

If you are moving software, you must move the license keys.

To move a node-locked or floating key:

  1. Return the license key on the old desktop.
    Note: To help prevent confusion between the license key update file for the old desktop and the license key file for the new desktop, complete the return process before you order license keys for the new desktop.
  2. Use Add/Remove Programs on the Windows Control Panel to remove the IBM Rational software from the old desktop or server.
  3. Install the IBM Rational software on the new desktop or server.
  4. Request a new license key file from AccountLink
  5. Import the license key on the new desktop or server.
  6. If you are moving floating keys, start the license server and give users the host name of the new license server.
