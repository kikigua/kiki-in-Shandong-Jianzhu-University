Frequently Asked Questions

The following sections provide answers to commonly asked questions.

Why do I see an 'Invalid Host' message when I start the program?

You received the following message when you started a licensed program:

Failed to check out a <product-name> key:

Invalid host

The hosted of this system does not match the hostid specified in the license file

In this case, you may either have moved your IBM Rational product from one computer to another without returning the licenses from the first computer, or imported the new license file to the wrong computer.

The permanent license key file is generated for a specific hostname and host ID. You can either import the file to the correct computer or move the license file to the computer that you are currently using.

How do I request a license key for a computer that does not have e-mail?

Use these steps to request a license for a system that does not have an HTTP connection.

  1. Find the your License Certificate.
  2. Print the license request fax form. To find the form, install the software and open the file in the Rational common directory (<Rational_directory>\common\License Fax Form - <language>.htm).
  3. Use the License Certificate to fill out the fax form.
  4. Send the request to Licensing Support.
  5. If you specified an e-mail address in the contact information, you will receive a license key file attached to an e-mail message.

Why doesn't the IBM Rational product that I installed work?

Ensure that you installed the product that you purchased by looking at your License Certificate.

Then, make sure you installed the permanent license file that you received from AccountLink on the correct computer. For other troubleshooting suggestions, see your product installation guide in the Rational common directory.
