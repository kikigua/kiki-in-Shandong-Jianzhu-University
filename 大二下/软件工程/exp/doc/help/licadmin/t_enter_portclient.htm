Entering port values in clients

Assigning a value to the port in your Rational License Key Administrator is necessary in firewall situations.

The following procedure assumes that you have entered the hostnames of the license servers in the License Key Administrator (LKAD) of your computer. Do not assign a value to this port unless your administrator instructs you to do so.

To enter port values from single, multiple or redundant servers:

  1. Ask your system administrator for the vendor port.
  2. Select Settings > Client/Server Configuration.
  3. In the Client/Server Configuration window, click the text box to the right of the vendor port box and enter the value.
  4. If there are other license servers, highlight the next license server and repeat Steps 1-3.
  5. Click OK.
