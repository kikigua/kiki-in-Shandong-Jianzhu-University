Entering license servers on clients

You can enter host names on your client for single, multiple, or redundant license servers.

The Client/Server Configuration window enables you to "point" your desktop at single, multiple, or redundant license servers to obtain floating licenses.

Note: To enter redundant servers, change the Server Type. Click Single next to Server Type. Select Redundant in the menu. Enter the Primary, Secondary, and Tertiary license server hostnames in the correct order. Press Tab to move to the next line.

To enter the servers' host names on your client:

  1. Contact your system administrator for the license server names.
  2. Launch Start > All Programs > IBM Rational > Rational License Key Administrator.
  3. Select Settings > Client/Server Configuration.
  4. Click Add Server.
  5. Enter the hostname in the Values column next to Server Name by clicking New-Server. Press Enter.
  6. If your system administrator has given you the hostnames of additional license servers, click Add Server and repeat Step 5.
  7. Click OK after you have entered all servers.
