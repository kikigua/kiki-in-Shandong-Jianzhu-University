Administrating IBM Rational Common Licensing

Administrators can use this information to understand and implement IBM Rational Common Licensing (powered by FLEXlm software).

  • Managing license keys
    With AccountLink on the Web you can request and return permanent node-locked, floating, and named-user floating license keys for products on Windows, Linux, and the UNIX system.
  • Configuring license servers
    If you are using floating licenses, you have the choice of configuring a single license server, multiple license servers, or a redundant server environment.
