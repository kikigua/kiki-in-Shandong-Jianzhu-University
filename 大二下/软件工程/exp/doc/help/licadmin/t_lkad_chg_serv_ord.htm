Changing the server search order

You can change the order in which IBM Rational products on your client desktop access license servers.

The license server list is displayed in the Client/Server Configuration window. The task of changing the server search order does not affect the order in which IBM Rational products request licenses from a server.

  1. In the Rational License Key Administrator, click Settings > Client/Server Configuration. The current server search order is displayed in the Search Order list.
  2. Click the Up and Down buttons to change the search order for selected servers in the list.
  3. Optionally, you can also add and remove servers from the list.
