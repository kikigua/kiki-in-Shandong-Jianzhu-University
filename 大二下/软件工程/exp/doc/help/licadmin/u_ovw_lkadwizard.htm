License Key Administrator wizard

The License Key Administrator wizard takes you through the process of:

  • Pointing your client to a license server for floating licenses.
  • Importing a permanent license key.
  • Entering a temporary license key.
  • Connecting to the AccountLink Web site to request your permanent license keys.

To prevent the wizard from appearing each time you start the License Key Administrator, select the Don't show this wizard on startup again... check box near the bottom of the window.

To access the wizard, click License Keys > License Key Wizard on the menu bar.
