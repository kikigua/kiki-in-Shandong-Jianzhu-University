Finding the hostname and host ID of a computer

The hostname and host ID are required information in AccountLink.

To get the hostname and host ID of a license server or desktop:

  1. Install the IBM Rational product on the computer.
  2. Open the License Key Administrator from the Start menu.
  3. Select License Keys > Host Data in the menu bar.
Note: AccountLink also provides a tool to help you find this information.
