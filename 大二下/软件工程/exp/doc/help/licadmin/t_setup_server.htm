Setting up single or multiple license servers

You can set up an environment with single or multiple license servers.

Use these instructions for single and multiple license servers. Do not use them to configure redundant license servers.

To set up a license server:

  1. Install the license server software on the server. For instructions, see the IBM Rational License Management Guide in the Rational common directory.
  2. Enter or import the license keys on the license server.
  3. To configure named-user floating licenses, edit the options file.
  4. Start the license server.
  5. Enter port values (optional).
  6. Give users the server name to enter into their License Key Administrators. This step is not necessary if the server name was entered into a site defaults file in a network release area. For more information about release areas, read your IBM Rational product installation guide.
