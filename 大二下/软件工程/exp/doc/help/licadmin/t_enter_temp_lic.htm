Entering temporary license information

Use the IBM Rational License Key Administrator (LKAD) to enter temporary license keys.

If you are entering a temporary floating key, you must first install the Rational license server software on the license server.

To enter information into the LKAD wizard:

  1. If the wizard does not launch after you install the product, select License Keys > Enter License Key in the menu bar. If you want to enter license keys on an Rational license server, install the server software first.
  2. Select the license type: node-locked or floating and named-user floating.
  3. In the next window, enter the following information:
    • Product
    • Expiration Date
    • License Key
    • Quantity (if you are installing a floating license key)
  4. Click Finish.
