Getting started with IBM Rational Common Licensing (powered by FLEXlm software)

IBM Rational Common Licensing (powered by FLEXlm software) is the licensing model supported by most IBM Rational products.

The IBM Rational License Key Administrator (LKAD) provides an interface to Rational Common Licensing features. The LKAD is installed with your IBM Rational software and with your license server software. Use the LKAD or the LKAD Wizard to enter or import license keys and to change your license configuration.

Note: You must have administrative privileges on the computer before you can enter or import license key information in the LKAD.

If you want to evaluate IBM Rational software, you need to get a temporary license key from your IBM sales representative.

If you have purchased the product, you need to get a permanent license key from AccountLink and install it on either a license server or desktop. You do not need to install the temporary license key.

If you were given an emergency license key, you must follow the instructions for installing a temporary license key.

  • License types
    IBM Rational Common Licensing (powered by FLEXlm software) provides three license types: floating, named-user floating, and node-locked.
  • License key types
    Floating, named-user floating, or node-locked licenses can be permanent and temporary keys.
