Starting the server on Windows XP

You can start the IBM Rational License Server on Microsoft Windows XP using this procedure.

You need to enter the temporary license information or import the floating license file first.
To start the license server on Windows XP:
  1. Click Start > Control Panel and double-click Administrative Tools > Services.
  2. Select FLEXlm License Manager and click Start the service.
