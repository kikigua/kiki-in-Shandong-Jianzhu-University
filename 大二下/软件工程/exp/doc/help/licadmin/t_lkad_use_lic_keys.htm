Importing and configuring license keys

You can use the IBM Rational License Key Administrator (LKAD) to import and implement license keys for licensed software.

  • Reviewing license information
    You can review information on node-locked, floating, and served licenses on the system that you are using.
  • Importing license keys
    You can use the Rational License Key Administrator to import license keys to a license server or desktop.
  • Entering temporary license information
    Use the IBM Rational License Key Administrator (LKAD) to enter temporary license keys.
  • Entering permanent license information
    You can configure a permanent license key for your IBM Rational product without a Rational license server.
  • Using node-locked licenses
    Use the License Key Administrator to enter temporary and import permanent license keys on the desktop. These keys do not require license servers.
  • Configuring clients to use floating license keys
    The license server manages a single pool of floating license keys. It grants one license key per client per product.
  • Configuring license usage order
    IBM Rational products request licenses in a specific order.
