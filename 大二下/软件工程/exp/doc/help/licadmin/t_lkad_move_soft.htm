Moving licensed software

If you are moving licensed software, you must move the license keys with the software.

Use AccountLink to return the license keys to your IBM Rational account. When you return a license key, you do not physically give the license key back to IBM. Instead, use the Return transaction in AccountLink. The AccountLink transaction updates the IBM Rational records to indicate that you are no longer using the product on that computer. This adjusts the count of registered products in your account.

Then AccountLink sends you a file to update the license key file on the old computer in accordance with the license agreement.

  1. Removing a license key
    You may want to return a license key; for example, when moving software from one computer to another, you return the original node-locked license key and then request the license key for the software on the new computer.
  2. Moving a license key
    If you are moving software, you must move the license keys.
