Entering port values on license servers

In some cases, you may need to change the port values on your single, multiple, or redundant license servers.

The default value of the lmgrd port in the license server is 27000. There are two situations in which you change the port value:

  • If another application uses the default TCP/IP port of 27000, change the lmgrd port value in all license servers.
  • If a firewall exists between the license server and the clients:
    1. Change the lmgrd port and add a vendor port value in all the license servers.
    2. Enter the vendor port value in all the clients.

These procedures assume that you have already started the license servers and specified the license server names in the clients.

To change the lmgrd port value on single, multiple, or redundant license servers:

  1. Select Settings > Server Ports in the License Administrator.
  2. In the Server Ports window, enter a value in the lmgrd Port. In a firewall situation, enter a value in the Vendor Port . Do not set the lmgrd port and the vendor port to the same values.
  3. Click OK.
  4. Repeat Steps 1-3 for the other license servers. The port numbers can be the same on all license servers.
  5. In the firewall situation, give users the vendor port value to enter into their clients.
