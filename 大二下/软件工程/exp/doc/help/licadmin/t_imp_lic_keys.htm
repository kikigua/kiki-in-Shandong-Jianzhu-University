Importing license keys

You can use the Rational License Key Administrator to import license keys to a license server or desktop.

To import a license key on a desktop or license server:
  1. Select License Keys > Import License Key(s) on the menu bar.
  2. In the Import License Key(s) window, find the *.upd or *.txt file and select it.
  3. Click Open.
  4. Click Import in the Confirm Import window.
