Returning license keys

You can use the AccountLink online licensing tool to return license keys to your IBM Rational account.

This transaction updates the IBM records to indicate that you are no longer using the software on the old system.
Note: To move a license key from one machine to another, the license key(s) must first be returned, then the "Get license key(s)" option can be used to register that license key or keys to the new machine.

To return license keys:

  1. Go to AccountLink.
  2. Click the Return license key(s) link.
  3. Complete the steps to provide your account number, host name, and other information.
If you want to move the license key to another system, you can now select the Get license key(s) option to specify the information for the new system.
