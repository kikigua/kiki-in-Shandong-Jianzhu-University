Managing license keys

With AccountLink on the Web you can request and return permanent node-locked, floating, and named-user floating license keys for products on Windows, Linux, and the UNIX system.

  • About AccountLink
    Rational AccountLink is a Web-based tool that manages your permanent license keys for products on Windows, Linux, and the UNIX system.
  • Accessing AccountLink on the Web
    You can access AccountLink through the IBM Web site.
  • Getting license keys
    You can use the AccountLink online licensing tool to get your IBM Rational license keys.
  • Returning license keys
    You can use the AccountLink online licensing tool to return license keys to your IBM Rational account.
  • Moving licensed software
    If you are moving licensed software, you must move the license keys with the software.
