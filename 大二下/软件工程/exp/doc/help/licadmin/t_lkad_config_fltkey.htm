Configuring clients to use floating license keys

The license server manages a single pool of floating license keys. It grants one license key per client per product.

  • Entering license servers on clients
    You can enter host names on your client for single, multiple, or redundant license servers.
  • Entering port values in clients
    Assigning a value to the port in your Rational License Key Administrator is necessary in firewall situations.
  • Changing the server search order
    You can change the order in which IBM Rational products on your client desktop access license servers.
