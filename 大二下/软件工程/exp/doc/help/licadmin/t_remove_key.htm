Removing a license key

You may want to return a license key; for example, when moving software from one computer to another, you return the original node-locked license key and then request the license key for the software on the new computer.

To return a node-locked or floating license key file:

  1. Use AccountLink to return the license keys to your IBM Rational account.
  2. Import the license key file that you receive from AccountLink on the old license server or desktop.
